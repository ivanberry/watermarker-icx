# 使用

-   @param container: 水印容器，默认为`body`
-   @param width
-   @param height
-   @param textAlign
-   @param textBaseline
-   @param font
-   @param fillStyle
-   @param content: 水印文本内容，默认为*iCarbonX*
-   @param rotate: 文本选址角度，默认为 30
-   @param zIndex

## 安装

```bash
npm install --save watermarker-icx
```
